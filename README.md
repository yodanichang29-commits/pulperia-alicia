# 🏪 Pulpería Alicia — Sistema de Gestión Completo

Sistema de gestión integral para pequeños negocios. Incluye punto de venta (POS), inventario, caja de ventas y gestión de proveedores en una sola aplicación.

---

## 📋 Módulos

| Módulo | Descripción |
|--------|-------------|
| **POS / Punto de Venta** | Pantalla de cobro rápida para atención al cliente |
| **Inventario** | Control de stock, entradas y salidas de productos |
| **Caja de Ventas** | Registro de transacciones y historial de ventas |
| **Proveedores** | Gestión de proveedores y productos suministrados |

---

## 💡 Descripción

Pulpería Alicia es una solución completa pensada para negocios pequeños que necesitan digitalizar sus procesos sin complicaciones. Integra en una sola plataforma todo lo necesario para operar: desde la venta al cliente hasta el control de lo que entra y sale del inventario y la relación con proveedores.

---

## ⚙️ Tecnologías

- **Backend:** PHP · Laravel
- **Base de datos:** MySQL
- **Frontend:** Blade Templates · CSS · JavaScript
- **Entorno local:** XAMPP

---

## 🚀 Instalación local

```bash
git clone https://github.com/yodanichang29-commits/pulperia-alicia.git
cd pulperia-alicia
composer install
cp .env.example .env
php artisan key:generate
# Configurar DB en .env
php artisan migrate
php artisan serve
```
